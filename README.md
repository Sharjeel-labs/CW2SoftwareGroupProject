## How to Run

1. Clone the repository
2. Install dependencies:
   pip install django
3. Run migrations:
   python manage.py migrate
4. Start server:
   python manage.py runserver 8080
5. Open:
   http://127.0.0.1:8080/

   ## Branching Strategy
Each team member works on feature branches (e.g. s3-messages) and merges via pull requests.
